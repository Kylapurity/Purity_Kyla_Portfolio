# Purity Kihiu — Portfolio

Personal portfolio website showcasing Zoho development work and software/UI projects.

🔗 Live: [purity-kyla-portfolio.vercel.app](https://purity-kyla-portfolio.vercel.app)

## Built With

- React
- Framer Motion (animations)
- React Icons
- Tailwind CSS / custom CSS

## Features

- **About** — intro, experience, and skills overview
- **Portfolio**
  - Filter by category: All, Zoho, Web, Mobile, Design
  - Zoho section with detailed case studies (role, description, outcome, skills)
  - Image carousels per project with click-to-expand lightbox view
- Fully responsive layout (desktop, tablet, mobile)

## Getting Started

```bash
npm install
npm start
```

Runs the app locally at `http://localhost:3000`.

## Build

```bash
npm run build
```

## Project Structure

```
src/
├── assets/          # images & screenshots
├── components/
│   ├── intro/       # About Me section
│   └── portfolio/   # Portfolio section (Zoho + project cards)
```
